Air Quality Widget

Provided by the Mid-Ohio Regional Planning Commission (MORPC) & the U.S. EPA AirNow program.

Review the README for more information on configuring the widget.
This example depends on live data. It requires an AirNow API key before it will load properly.



Compact Widget

This is the compact widget. It is best used in the header, footer, or on the sidebar. However, it can be displayed right within the main page. It can be configured to display the forcast when clicking the ⓘ icon.

Place the following HTML where you want the compact widget to appear: 
<div id="aqi-widget-container-compact"></div>
